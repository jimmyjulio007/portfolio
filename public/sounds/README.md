# Sound Files Directory

Place your audio files here:

- `hover.mp3` - UI hover sound effect
- `click.mp3` - Button click sound effect  
- `transition.mp3` - Page transition sound
- `ambient.mp3` - Background ambient sound (optional)

Recommended formats:
- MP3 (best browser support)
- File size: Keep under 50KB for UI sounds
- Volume: Normalize to -6dB to -3dB

You can find free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://pixabay.com/sound-effects/
