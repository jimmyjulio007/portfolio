# 3D Models Directory

This directory contains 3D assets for the portfolio.

## Supported Formats
- **GLB** (preferred): Binary glTF format, optimized for web
- Texture resolution: 1k recommended for performance

## Usage
Place your `.glb` files here and they will be accessible at `/models/filename.glb`

## Performance Tips
- Keep models under 10MB
- Use 1k textures for optimal loading
- Compress geometry when possible
- Use Draco compression for large models

## Example Files
- `hero-model.glb` - Main hero section 3D model
- `about-avatar.glb` - About section character/avatar
